# React Portfolio Website

A personal portfolio website built with **React** and **Bootstrap** to showcase my projects, skills, and experience. The website is fully responsive, clean, and designed to highlight real-world projects in data analytics, machine learning, and web development.

## Features

- Home section with introduction and headline
- About section highlighting education, internships, and skills
- Projects section showcasing deployed apps and GitHub links
- Contact section with email, GitHub, and LinkedIn links
- Responsive design for mobile, tablet, and desktop
- Built using React components and Bootstrap for styling

## Technologies Used

- React
- Bootstrap
- HTML & CSS
- JavaScript
- Git & GitHub for version control
